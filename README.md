# Playwright Automation Framework

This repository contains UI automation tests built using **Playwright with JavaScript**.

## Tech Stack

* Playwright
* JavaScript
* Node.js
* GitHub

## Project Structure

```
tests/
   Ulbasictest.spec.js

playwright.config.js
package.json
```

## How to Run Tests

Install dependencies:

```
npm install
```

Run tests:

```
npx playwright test
```

View report:

```
npx playwright show-report
```

## Test Scenario

Automated UI tests for the Sauce Demo application using Playwright.
